# Essence of Art Gallery - E-commerce Website

A fully functional e-commerce website for an art gallery built with PHP, MySQL, Bootstrap.

## Features

### Admin Panel
- Add new artworks with images
- Manage artworks (edit, delete)
- View customer orders
- Track order status

### Customer Side
- Browse artworks with filters and search
- View artwork details
- Shopping cart functionality
- Secure checkout process
- Order confirmation

## Technologies Used
- **Backend:** PHP, MySQL
- **Frontend:** HTML, CSS, Bootstrap 5
- **Database:** MySQL with PDO
- **Session Management:** PHP Sessions
